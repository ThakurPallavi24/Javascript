What is callback ?

A callback is a function that is passed as an argument to another function. Purpose of a callback is to handle asynchronous operations and enable non-blocking behaviour in programming.

What is callback hell?

It is a situation where multiple nested callbacks are used to handle asynchronous operations.

Why we use async , await?

We use async/await to make asynchronous code in JavaScript easier to read and write. async function always return a promise.The await keyword is used inside an asynchronous function to pause the execution until a Promise is resolved or rejected.

What is promise?

In JavaScript, a promise is a built-in object. Promise is a sollution to callback hell. We use promise constructor to create a promise which takes an executor function as an argument.

What is DOM?

DOM stands for document object model. It represents the structure of HTML and XML object as a tree of object. Basically it is a programming interface provided by web browsers.
